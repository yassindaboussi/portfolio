## ⚡ Overview

A **professional portfolio website** that demonstrates modern web development practices. Built entirely with **TypeScript** for type safety, featuring responsive design, smooth animations, and optimal performance.

**Live:** https://yassindaboussi.netlify.app/

---

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| **Next.js 16** | React framework with SSR & SSG for optimal performance |
| **TypeScript** | Type-safe development (95.9% of codebase) |
| **React 19** | Modern UI components and hooks |
| **Tailwind CSS 4** | Utility-first styling with PostCSS |
| **Framer Motion** | Smooth animations and page transitions |
| **Lucide Icons** | Beautiful, lightweight icon library |

## 🚀 Quick Start

### Prerequisites
```bash
Node.js 18+ | npm or yarn
```

### Setup

```bash
# Install dependencies
npm install

# Development server
npm run dev
# Open http://localhost:3000

# Production build
npm run build
npm start

# Linting
npm run lint
```
